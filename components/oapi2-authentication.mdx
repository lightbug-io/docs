# Authentication

Authorization uses JWT Bearer tokens generated from the [`/users/login`](#?route=post-/users/login) endpoint.

You can find an overall summary of API authentication [here](/apis/authentication).

Below you can enter a token to use with the API while browsing this documentation.

Tokens from both the v1 and v2 API can be used here.
