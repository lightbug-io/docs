# Identity

These settings can be used to identify, filter, and spot your device more easily.

## Name

Name of the device for easy recognition in apps and APIs.

## Color

The color to use for the device in various interfaces.

For example, the [cloud app](/apps/cloud) colours the map dots with this colour.

## Tags

Devices can have one or more tags applied to them.

Some tages can be automatically applied based on zonal conditions.
