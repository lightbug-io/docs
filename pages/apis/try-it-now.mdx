# Try it now

The navigable documentation specifications  for both API versions provide a `Execute` buttons that can be used to test the APIs directly from the documentation.

![](https://i.imgur.com/9fvEOSr.png)

The `Execute` button will automatically add the `Authorization` header to the request based on the token provided in each documentation specification authentication page.

![](https://i.imgur.com/jnbJshA.png)

The full `curl` example request, including token, will be visible once the `Execute` button is pressed.

![](https://i.imgur.com/NFXo6Qj.png)
