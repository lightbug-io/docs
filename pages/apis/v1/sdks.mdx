# SDKs

You can find some auto generated SDKs for the Version 1 API in various languages [on Github](https://github.com/remotethings).

 - [Javascript (Node)](https://github.com/remotethings/tracking-api-node)
 - [Javascript (React)](https://github.com/remotethings/tracking-api-react)
 - [PHP](https://github.com/remotethings/tracking-api-php)
 - [Python](https://github.com/remotethings/tracking-api-python)
 - [C#](https://github.com/remotethings/tracking-api-csharp)

There are currently no SDKs for the Version 2 API.
